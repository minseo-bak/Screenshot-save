디스코 엘리시움풍 저장화면 플러그인을 제작하기 위한 레포지토리.

1. ALTsavescreen 플러그인을 기반으로 위치를 스왑한다 
2. 플레이 시간 등 세이브 내용이 출력되던 곳에 스크린샷 출력을 삽입한다.
3. 블록형 저장공간을 목록형으로 만든다.

A repository for creating Disco Elysium-style save screen plugins.

1. Swap location based on ALTsavescreen plugin
2. Insert a screenshot output where save information such as play time is displayed.
3. Convert block-type storage space into list-type.
